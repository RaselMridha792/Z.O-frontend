3-Week Sprint Structure:
Week 1: Setup & Authentication System
Frontend: Next.js, Tailwind, Redux, basic layout, and authentication UI (signup, login, OAuth).


Backend: Express.js, PostgreSQL setup, user authentication (JWT, OAuth), email verification.


Week 2: Quiz System, Payment, and Leaderboard
Frontend: Quiz UI, timer, scoring display, leaderboard page.

Backend: Quiz logic (CRUD API for quizzes, questions, scoring), payment integration (bKash API), leaderboard logic.


Week 3: Admin Panel, QA, and Deployment
Frontend: Admin dashboard, quiz/user management UI, testing & bug fixing.

Backend: Admin routes, finalize API endpoints, DevOps setup, CI/CD for production.
